# Images folder

Drop these five files here. The HTML references them by these exact filenames.

- `hero-clinician.jpg` — main hero photo (clinician + patient + device, wide)
- `skin-diagram.png` — skin tissue cross-section with laser beam
- `device.png` — product shot of the Noema-Laser device
- `clinician-closeup.jpg` — clinician working on patient (vertical-ish crop preferred)
- `clinic-exterior.jpg` — optional, modern clinic photo

Until you add them, the site shows clean placeholder blocks. The layout works either way.
